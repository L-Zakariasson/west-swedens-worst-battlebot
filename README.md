This is a school project for my intergrated systems course.
Using C and the atmega 328p processor i am creating a very shitty battlebot.

The hardware incluedes a HC-06 bluetootth module, a 5 to 3,3 V logic converter, and the Motron M3T453 i2c motor controller.
it takes commands from a ps5 controller using a computer runninga python scrop to send commands over BT serial and the bot uses  that to controll two motors to provide "Tank" stering and one motor for the weapon.

The body is built using Lego.
