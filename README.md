This is a school project for my integrated systems course.
Using C and the atmega 328p processor I am creating a very shitty battlebot.

The hardware includes a HC-06 bluetooth module, a 5 to 3,3 V logic converter, and the Motron M3T453 i2c motor controller.
It takes commands from a ps5 controller using a computer running a python script to send commands over BT serial and the bot uses  that to control two motors to provide "Tank" steering and one motor for the weapon.

The chassi is built using Lego.
